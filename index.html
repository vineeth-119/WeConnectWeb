About WeConnect

WeConnect is a full stack application used by police and public that can act as an interface between the both. Public can lodge an online complaint which is recorded by his Aadhar Number, can apply for NOC certificate which is generally used for applying to foreign universities or other Govt. Exams, can check if the person nearby is a criminal or not by using Face Recognition algorithm. These functionalities are provided in an Android Application.

For Police, it has a client side web application that needs custom login activity. For each police station, a unique UserID and password are provided with which they can manage all requests from public and also manage criminal record or database.
